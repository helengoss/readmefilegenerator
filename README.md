# README file generator 


## Description
This CLI (command line application) will create a README file for you, prompting you with questions for what you want in your REDAME file, within the terminal, then create the file (or overwrite the existing readme)

## Installation
Clone the repository
Ensure you are using python version 3.7 or above
Create activate a virtual environment (python -m venv myenv)
Install dependencies (pip install -r requirements.txt)
Install InquirerPy, works cross platform for all OS. Although Unix platform may have a better experience than Windows. (pip install -m InquirerPy)
Python main.py

## Usage
Running python3 main.py
